## 简介

为即梦 Seedance 2.0 撰写高质量视频生成提示词的完整工程体系。融合 lanshu-awesome-ai-video-kit 的 8 要素工程化公式和 dexhunter/seedance2-skill 的 12 场景实战模板。

## 核心模块

| 模块 | 内容 |
|------|------|
| 8要素公式 | 主体→动作→场景→光影→运镜→风格→画质→约束 |
| 输入约束 | 图片≤9/视频≤3/音频≤3/总文件≤12，各格式限制 |
| @引用系统 | 首帧/尾帧/人物/场景/运镜/动作/特效/音频引用语法 |
| 运镜词典 | 基础运镜(推拉摇移跟) + 高级运镜(希区柯克/鱼眼/机械臂) + 景别 |
| 12场景模板 | 人物一致性/运镜复刻/特效复刻/视频延长/视频编辑/音乐卡点/一镜到底/电商/短剧/科普等 |
| 情绪外化表 | 6 种情绪的动作替代方案 |
| 关键帧驱动 | 首帧+尾帧双锁 + 运动 + 时长 + 约束 |
| TTS先行 | 剧本→TTS→时长→视频参数 |
| 成品模版库 | 产品广告/短剧/舞蹈/风光/仙侠/科普 |
| 避坑清单 | 12 条常见错误与对策 |
