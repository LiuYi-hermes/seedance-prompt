# Seedance 2.0 视频提示词 v4.2

## 8要素工程化公式（核心骨架）

```
精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质 + 约束条件
```

| 要素 | 填写 | 示例 |
|------|------|------|
| 精准主体 | 谁/什么 | 一位25岁亚洲女性，黑色长发 |
| 动作细节 | 做什么/怎么做 | 缓慢将手伸向后背，手指微微颤抖 |
| 场景环境 | 在哪/什么氛围 | 深夜办公室，仅有显示器蓝光 |
| 光影色调 | 主光+色温 | 顶部冷白荧光为主，屏幕蓝光补面 |
| 镜头运镜 | 景别+运动 | 中近景固定，缓推至面部特写 |
| 视觉风格 | 电影感/写实 | 电影级写实，4K，浅景深 |
| 画质 | 技术指标 | 毛孔清晰，皮肤纹理真实，无过度磨皮 |
| 约束条件 | 不要什么 | 无字幕，无水印，无Logo，无畸变 |

## 输入约束（Seedance 2.0 硬限制）

| 输入类型 | 数量上限 | 支持格式 | 大小限制 |
|----------|----------|----------|----------|
| 图片 | ≤ 9 张 | jpeg/png/webp/bmp/tiff/gif | 每张 < 30 MB |
| 视频 | ≤ 3 个 | mp4/mov | 每个 < 50 MB，总时长 2–15s |
| 音频 | ≤ 3 个 | mp3/wav | 每个 < 15 MB，总时长 ≤ 15s |
| **总文件** | **≤ 12 个** | — | — |

**输出**: 4–15秒，480p–720p，自带音效/配乐
**红线**: 禁止写实真人脸部素材（系统拦截），参考视频生成费用略高

## @引用系统（完整版）

### 引用方式

```
@图片1    @图片2    @图片3   ...    (最多9张)
@视频1    @视频2    @视频3          (最多3个)
@音频1    @音频2    @音频3          (最多3个)
@角色四视图  @首帧参考图  @尾帧参考图  @TTS音频
```

### 角色分配表

| 用途 | 写法 |
|------|------|
| 首帧 | `@图片1 作为首帧` |
| 尾帧 | `@图片2 作为尾帧` |
| 人物形象 | `参考 @图片1 的人物形象` |
| 场景/背景 | `场景参考 @图片3` |
| 运镜 | `参考 @视频1 的运镜效果` |
| 动作 | `参考 @视频1 的动作编排` |
| 特效/转场 | `完全参考 @视频1 的特效和转场` |
| 节奏/节拍 | `视频节奏参考 @视频1` |
| 音色/语气 | `旁白音色参考 @视频1` |
| 背景音乐 | `BGM参考 @音频1` |
| 音效 | `音效参考 @视频3 的音效` |
| 服装 | `穿着 @图片2 的服装` |
| 产品外观 | `产品细节参考 @图片3` |
| 字体/文字 | `字体参考 @图片2 的字体` |

### 多引用组合

```
@图片1 的人物作为主体，参考 @视频1 的运镜和动作编排，
BGM参考 @音频1，场景参考 @图片2
```

## 运镜词典

### 基础运镜
| 术语 | 说明 | 英文 |
|------|------|------|
| 推镜头/慢推 | 镜头向主体靠近 | Push in / Slow push |
| 拉镜头/后拉 | 镜头远离主体 | Pull back / Pull away |
| 左摇/右摇 | 镜头水平旋转 | Pan L/R |
| 上摇/下摇 | 镜头垂直旋转 | Tilt Up/Down |
| 跟随镜头/跟拍 | 镜头跟随主体移动 | Track / Follow shot |
| 环绕镜头 | 镜头围绕主体旋转 | Orbit / Revolve |
| 一镜到底 | 全程无剪辑连续镜头 | One-take / Oner |
| 固定镜头 | 静止不动（默认首选） | Static |

### 高级运镜
| 术语 | 说明 |
|------|------|
| 希区柯克变焦 | 推拉+变焦眩晕效果 |
| 鱼眼镜头 | 超广角畸变镜头 |
| 低角度仰拍 | 低机位向上，英雄感 |
| 俯拍/鸟瞰 | 从高处向下 |
| 第一人称主观视角 | 从角色视角出发 |
| 快速摇镜 | 极速水平旋转产生运动模糊 |
| 机械臂跟随 | 多角度灵活跟随人物视线 |

### 景别
| 术语 | 英文 | 说明 |
|------|------|------|
| 大远景 | ELS | 展示完整环境 |
| 全景 | FS | 完整人物 |
| 中景 | MS | 腰部以上 |
| 中近景 | MCU | 头部和肩部 |
| 面部特写 | CU | 人脸充满画面 |
| 大特写 | ECU | 仅眼睛/嘴巴等细节 |

## 情绪外化表

**禁止写「很悲伤」「非常愤怒」——用动作替代情绪：**

| 情绪 | ❌ 抽象 | ✅ 外化动作 |
|------|---------|-----------|
| 悲伤 | 很悲伤 | 低头，嘴唇轻颤，眼泪悬在眼眶未落下 |
| 愤怒 | 非常愤怒 | 下颌紧绷，握拳至指节发白，呼吸急促 |
| 恐惧 | 害怕 | 瞳孔骤缩，后退半步，手扶墙壁寻找支撑 |
| 喜悦 | 很开心 | 眼角弯起，嘴角无法控制地上扬，深吸一口气 |
| 惊讶 | 震惊 | 眉毛上挑，嘴巴微张，身体后倾 |
| 紧张 | 紧张不安 | 反复摩挲手指，目光游移，喉结上下滚动 |

## 关键帧驱动

### 双链路决策
| 链路 | 场景 | 控制力 |
|------|------|--------|
| 快速I2V | 简单对话/特写/固定镜头 | 弱 |
| 关键帧插值 | 动作戏/转场/情绪高潮/运镜复杂 | 强 |

### 关键帧模板（8要素增强版）
```
【首帧】: 主体(谁+在哪) + 动作(起始状态) + 场景 + 光影 + 镜头 + 风格 + 画质 + 约束
【尾帧】: 主体(最终位置) + 动作(结束状态) + 场景变化 + 光影连续 + 镜头(运动终点)
【运动】: {从运镜词典选}: 推/拉/摇/移/跟/升/降
【时长】: {TTS实际时长}s
【约束】: L1-L3死锁，角色锁于四视图，无变形，光影连续
```

最后：血流速度减慢，血管腔变窄，灯光转暗暗色调。教育警示氛围。15秒。

## 避坑清单（融合版）

| # | 坑 | 对策 |
|---|-----|------|
| 1 | **人物漂移** | L1-L3死锁+四视图引用，防止「演着演着换人了」 |
| 2 | **双胞胎** | 多人场景标清角色位置+服饰差异 |
| 3 | **字幕/水印** | 约束条件必加「无字幕/无水印/无Logo」 |
| 4 | **风格漂移** | 首帧+尾帧双锁，防止中间帧风格突变 |
| 5 | **画质劣化** | 真实感关键词 + REAL-Video-Enhancer 后处理 |
| 6 | **嘴型不对** | video-retalking 后期校准 + echomimic 前期控制 |
| 7 | **引用模糊** | 不能说「参考@视频1」，必须说清参考什么（运镜？动作？特效？） |
| 8 | **指令冲突** | 不能同时要「固定镜头」和「环绕镜头」 |
| 9 | **内容过载** | 不在4-5秒内塞入太多场景 |
| 10 | **素材无归属** | 上传N张图，每张都必须@标注用途 |
| 11 | **忽视音频** | 音效设计能大幅提升质量，必须写音频指导 |
| 12 | **时长不匹配** | 提示词复杂度必须匹配生成时长 |

## 交互流程

1. **明确目标**：广告？短剧？MV？科普？Vlog？
2. **清点素材**：有几张图？几个视频？几个音频？
3. **分配角色**：每个素材@标注清晰用途
4. **构建提示词**：
   - 开头设定主体和场景
   - ≥8秒用分时段描述
   - 明确运镜语言（默认固定）
   - 加入音频/音效设计
   - 添加风格修饰词
5. **检查约束**：总文件 ≤ 12、无写实人脸、时长在限制内
6. **优化打磨**：消除歧义，确保每个@有明确作用
